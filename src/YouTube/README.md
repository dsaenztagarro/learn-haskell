Companion modules to selected YouTube lectures. Useful for a second
pass over the same Stage 06 material covered by `Exts/Kinds/`.

| Module | Lecture | Link |
| --- | --- | --- |
| `Kinds`  | Tweag (Richard Eisenberg), *An introduction to Haskell's kinds* | https://www.youtube.com/watch?v=JleVecHAad4 |
| `Kinds2` | Tweag (Richard Eisenberg), *Getting a little fancy with Haskell's kinds* | https://www.youtube.com/watch?v=Qy_yxVkO8no |

lecture, not idiomatic application code. Read after the
