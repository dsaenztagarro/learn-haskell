# `Exts/Records/` — Record-syntax sugar

| Module | Extension | One-line idea |
| --- | --- | --- |
| `RecordPuns` | `NamedFieldPuns` | `C { a, b }` shorthand for `C { a = a, b = b }`. |
| `RecordWildCards` | `RecordWildCards` | `{..}` binds every field by name. |

Read in this order; `RecordWildCards` builds on the puns idea.
